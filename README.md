## Что делает программа

Пошаговый конвейер обработки данных:

| Этап | Описание |
|:---|:---|
| Загрузка CSV | Данные загружаются в массив и связный список |
| Фильтрация | Удаление выбросов по правилу трёх сигм (линейный поиск) |
| Сортировка | Сортировка слиянием (рекурсия) |
| Медиана | Поиск медианы по отсортированному массиву |
| Префиксные суммы | Построение массива накопленных сумм |
| Гистограмма | Текстовая гистограмма с рекурсивным разбиением диапазонов |
| Откат | Стек сохраняет состояние после каждого этапа — можно вернуться назад |
